# calc
Quick calculater

Calc is a simple command-line calculator written in C.  It has no dependencies other than standard C functions.

Here's an example session
```
$ calc

                 **********************************
                 *  CALC - The Simple Calculator  *
                 **********************************

Type HELP for help.

calc> a=120*5
600.000000

calc> b=30^2
900.000000

calc> a+b
1500.000000

calc> quit
```
